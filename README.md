## Projects

Side projects I tinker with on nights and weekends — mostly AI-native, Claude Code–powered, built around real workflows I run myself.

- 📓 **[vm-notes](https://github.com/vishalpmittal/vm-notes)**  
  Personal knowledge base for technical notes on AI/ML, system design, infra, and leadership. Driven by a `/add-blog` Claude Code skill that fetches articles, auto-categorizes, downloads images, and merges into topic notes.

- 🎯 **[vm-pats](https://github.com/vishalpmittal/vm-pats)**  
  Personal Application Tracking System with AI-powered resume review and tailored resume generation. Glass-UI Vite + Express app driven by the `claude` CLI; tracks gaps across applications and versions tailored resumes per role.

- 🔖 **[vm-bookmark-manager](https://github.com/vishalpmittal/vm-bookmark-manager)**  
  Local-first bookmark manager that runs entirely in the browser via the File System Access API — no server, no account, no cloud. Bookmarks live in two plain CSV files on disk that you own and control.

- 🧪 **[vm-interview-review](https://github.com/vishalpmittal/vm-interview-review)**  
  Standardized templates and Claude Code skills (`/review-candidate`, `/review-the-review`) for a 7-round hiring pipeline. Every round scores candidates against 9 core values with structured rubrics and an AI second-opinion pass.

- 🔑 **[vm-git-setup-multiple-identities](https://github.com/vishalpmittal/vm-git-setup-multiple-identities)**  
  Interactive shell script that wires up multiple git identities (work + personal) using SSH host aliases and git's `includeIf` directive. Run once, never accidentally mix commits again.

- 📋 **[vm-enterprise-ready-servicebook](https://github.com/vishalpmittal/vm-enterprise-ready-servicebook)**  
  Multi-tenant SaaS that turns the 14-pillar enterprise readiness checklist into a live, fillable, auditable application. React + Express + PostgreSQL on GKE; replaces the Confluence + Jira + spreadsheet + Slack sprawl most teams use today.

- 📊 **[vm-resume-rank](https://github.com/vishalpmittal/vm-resume-rank)**  
  Claude Code skill (`/rank-resumes`) that scores candidate resumes against a JD on 9 rubric parameters with one-line justifications, bonus points, and re-rank-with-custom-weights support.
